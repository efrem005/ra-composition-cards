Карточки
===

Вам необходимо реализовать компонент карточек, позволяющий использовать себя следующим образом:

![](./assets/card1.png)

![](./assets/card2.png)

В качестве CSS вы можете использовать Bootstrap 5, а подглядеть генерируемую разметку можете на [странице](https://getbootstrap.com/docs/5.3/components/card/).

Стили и пример разметки вы можете найти в папке [markup](./markup). Разметка дана для примера, вы можете реализовать её самостоятельно.

**Обязательно** необходимо реализовать карточки, показанные на изображениях выше. Дополнительные варианты карточек из примеров можно реализовать опционально.

Подсказка: используйте для этого `props.children` и `props` для отображения картинки.
